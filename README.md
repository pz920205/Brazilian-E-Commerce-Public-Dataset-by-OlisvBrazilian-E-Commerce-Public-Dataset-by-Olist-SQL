##  Valor del Proyecto
Este repositorio centraliza scripts avanzados de SQL diseñados para procesos de **ETL** y análisis de rentabilidad. El objetivo es demostrar capacidad técnica alineada con el pensamiento analítico y enfoque en negocio.

##  Stack Tecnológico
* **SQL:** Consultas complejas, Joins y vistas.
* **Python:** Procesamiento y modelado de datos.
* **Power BI:** Visualización de KPIs estratégicos.
* **Excel:** Análisis rápido y reporting.

##  Capacidades
*  **Enfoque en negocio:** Identificación de métricas de rentabilidad.
*  **Pensamiento analítico:** Resolución de problemas mediante datos.
*  **Transformación de datos:** Limpieza y estructuración eficiente.
